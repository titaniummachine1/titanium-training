# Training experiments

Non-production research scripts. **Do not import from production code** (`titanium_training/`).

| Subfolder | Contents |
| --------- | -------- |
| `lmr/` | Reduction-head training and counterfactual collection |
| `features/` | Field-plane extensions, color rotation, probes |
| `evaluation/` | Checkpoint comparison, pressure-source analysis |

Oracle bundles exclude this tree unless explicitly requested.
